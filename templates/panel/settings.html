Autenticação

Use Basic Auth com as credenciais configuradas nas variáveis de ambiente:

  • PAGARME_WEBHOOK_BASIC_AUTH_USER — usuário HTTP Basic Auth

No Pagar.me, acesse Configurações → Webhooks e cadastre a URL acima com o método POST e formato JSON.
