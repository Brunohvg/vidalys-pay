# Prompt — aplicar o kit visual corrigido no Vidalys Pay

Use exclusivamente os ativos deste pacote. Não redesenhe, não altere proporções e não gere novas versões por IA.

## Regras
- Use `brand/logo-horizontal.svg` no cabeçalho sobre fundo claro.
- Use `brand/logo-horizontal-negative.svg` sobre fundo azul-escuro.
- Use `brand/logo-symbol.svg` em navegação compacta e sidebar recolhida.
- Use `favicons/favicon.svg` como favicon vetorial, com PNGs de fallback.
- Use `pwa/app-icon.svg` no manifest e atalhos.
- Use `pwa/app-icon-maskable.svg` para `purpose: maskable`.
- Use `social/og-image.svg` como origem da imagem Open Graph.
- Use `splash/splash-screen.svg` na tela inicial da PWA.
- Use os ícones de `ui-icons/` sem misturar outras famílias visuais.
- Importe `docs/design-tokens.css` e use seus tokens em toda a interface.

## Tipografia
Use Inter. Hierarquia recomendada:
- Título principal: 28–32px, 700.
- Título de seção: 20–24px, 700.
- Corpo: 15–16px, 400.
- Labels: 13–14px, 600.
- Valores monetários: 28–36px, 700, numerais tabulares.

## Limites
Não altere regras de negócio, Pagar.me, Evolution API, webhooks, rotas, models ou validações. Corrija somente identidade visual, layout, responsividade, acessibilidade e uso dos assets.
