# Vidalys Pay — Assets profissionais v3

Este pacote substitui os assets anteriores.

## Diferença desta versão
- Um único símbolo canônico em todos os arquivos.
- SVGs reais, não imagens de prancha.
- Texto do logotipo convertido em curvas vetoriais.
- Exportações PNG geradas a partir dos SVGs.
- Favicon, PWA, Open Graph, splash e ícones de interface organizados.

## Pastas
- `brand/`: logotipos e símbolo.
- `favicons/`: favicon SVG, ICO e PNG.
- `pwa/`: ícones comuns e maskable.
- `social/`: imagem Open Graph.
- `splash/`: tela de abertura.
- `ui-icons/`: ícones funcionais SVG.
- `docs/`: manifest, tokens CSS e prompt de aplicação.
- `preview/`: prancha de conferência.

Os arquivos de fonte não fazem parte do pacote.
